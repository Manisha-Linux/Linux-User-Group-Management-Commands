# Linux-User-Group-Management-Commands
🐧 A collection of important Linux User and Group Management commands used in Linux Administration.

## Topics Covered

- User Creation
- Password Management
- Group Creation
- Adding Users to Groups
- Removing Users from Groups
- User Lock and Unlock
- Username and Group Name Modification
- Password Aging Policies
- User and Group Information Commands
- User and Group Deletion

## Commands Sheet

This repository contains a visual command sheet covering commonly used Linux User and Group Management commands.

- useradd
- passwd
- groupadd

- gpasswd
- groupmod
- chage
- userdel
- groupdel
- id
- groups
- who
- w
- last

## Purpose

The purpose of this repository is to document and revise Linux Administration concepts while preparing for Linux Administrator and System Administrator roles.
